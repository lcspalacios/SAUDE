O script faz a leitura de um arquivo de BPA-I e localiza inconsistências de procedimento 0203010086 para idades fora da faixa de 25 a 64 anos. 
Caso seja encontrado esse tipo de registro o procedimento é alterado para 0203010019 sendo então calculada uma nova chave verificadora para o arquivo.
Ao final é solicitado ao usuário que escolha um nome para salvar o novo BPA-I
